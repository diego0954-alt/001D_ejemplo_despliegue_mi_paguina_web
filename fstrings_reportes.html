// 01 · Fundamento

¿Qué son los F-Strings?

Los f-strings (formatted string literals) son la forma más poderosa y moderna de formatear texto en Python (desde la versión 3.6). Permiten incrustar expresiones directamente dentro de un string y controlar con precisión cómo se muestran los datos: ancho, alineación, decimales, relleno y más.

sintaxis_basica.py 
# Forma clásica (antigua, verbosa)
nombre = "María"
precio = 1234.5
print("Hola %s, tu total es %.2f" % (nombre, precio))

# Con .format() — mejor, pero aún limitado
print("Hola {}, tu total es {:.2f}".format(nombre, precio))

# F-string — conciso, potente, legible ✓
print(f"Hola {nombre}, tu total es {precio:.2f}")

# Incluso soporta expresiones directamente
items = [10, 25, 8]
print(f"Total de items: {len(items)}, Suma: {sum(items)}")
Sintaxis general:
f"texto {variable:[[fill]align][width][.precision][type]} más texto"
// 02 · Redondeo

Control de Decimales

El especificador :.Nf permite controlar exactamente cuántos decimales mostrar. Esencial para precios, porcentajes y datos científicos.

decimales.py 
precio = 1234.56789
iva     = 0.19
total   = precio * (1 + iva)

print(f"Precio bruto : ${precio:.2f}")        # 2 decimales
print(f"IVA (19%)    :  {iva * 100:.1f}%")      # 1 decimal + %
print(f"Total a pagar: ${total:.2f}")          # 2 decimales
print(f"Sin decimales: ${total:.0f}")          # 0 decimales
print(f"Notación cien.:  {total:.2e}")         # científica
$ python decimales.py

// 03 · Alineación

Alineación de Texto

El especificador de alineación define cómo se posiciona el contenido dentro de un campo de ancho fijo. Clave para columnas perfectamente alineadas en reportes de terminal.

< · Izquierda

Alinea el contenido a la izquierda del campo. Es el valor por defecto para strings.

f"{valor::<20}"

> · Derecha

Alinea el contenido a la derecha del campo. Por defecto para números.

f"{valor::>20}"

^ · Centro

Centra el contenido dentro del campo. Ideal para encabezados y títulos.

f"{valor::^20}"
alineacion.py 
producto = "Café Molido"
precio   = 4990

# Ancho 25, alineación izquierda
print(f"|{producto:<25}|")    # |Café Molido              |

# Ancho 25, alineación derecha
print(f"|{producto:>25}|")    # |              Café Molido|

# Ancho 25, centrado
print(f"|{producto:^25}|")    # |       Café Molido       |

# Combinar: descripción izquierda + precio derecha
ancho_desc  = 25
ancho_precio = 10

print(f"{producto:<25}${precio:>9,}")

// 04 · Relleno de Caracteres

Fill Characters (Relleno)

El carácter de relleno (fill) reemplaza los espacios vacíos al alinear. Se especifica antes del operador de alineación y puede ser cualquier carácter.

Sintaxis: f"{valor:FILLALIGNWIDTH}"

Donde FILL = cualquier carácter (espacio, -, *, =, ·, etc.)
ALIGN = < > ^
WIDTH = número entero (ancho total del campo)
relleno.py 
titulo  = "RESUMEN"
precio  = 9990.5
w       = 40   # ancho de la línea

# Relleno con guiones — muy común en reportes
print(f"{titulo:-^40}")       # ---------RESUMEN---------

# Relleno con igual — para secciones
print(f"{'TOTAL':=^40}")        # ==================TOTAL==================

# Relleno con asterisco — énfasis
print(f"{'ERROR':*^40}")        # *******************ERROR*******************

# Relleno con punto — para precios en menú
concepto = "Subtotal"
print(f"{concepto:·<30}${precio:>9.2f}")

# Ceros a la izquierda en números
num_factura = 42
print(f"Factura N°{num_factura:0>6}")   # Factura N°000042

Anatomy of an F-String Format Spec

Todos los elementos del especificador de formato se pueden combinar. Aquí el mapa completo:

anatomia_completa.py 
# {variable : [fill][align][sign][width][grouping][.precision][type]}
#              ──── ─────  ──── ───── ──────────  ──────────  ────
#              ·    ^      +    20    ,           .4          f

val = -1234567.891
print(f"{val:·^+25,.2f}")
# ····-1,234,567.89·····

# Un ejemplo completo de reporte en terminal
W = 45   # ancho total

print(f"{'':=^{W}}")                              # ═══════
print(f"{'TICKET DE COMPRA':^{W}}")               # título
print(f"{'':=^{W}}")
print(f"{'Producto':<28} {'Cant':^5} {'Precio':>10}")
print(f"{'':–^{W}}")
productos = [
    ("Café Molido 250g", 2, 3490),
    ("Leche Entera 1L",  3, 890),
    ("Pan Integral",     1, 1250),
]
for nombre, cant, precio in productos:
    subtotal = cant * precio
    print(f"{nombre:<28} {cant:^5} ${subtotal:>9,}")
